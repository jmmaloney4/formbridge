# ADRs (Architecture Decision Records)

This directory contains ADRs for the `jmmaloney4/formbridge` repository (fork of `nilsyai/formbridge`).

- Location: `docs/internal/decisions/`
- Template: `000-adr-template.md`
- Naming: `ADR NNN: Title` in the document; filename `NNN-kebab-title.md`
- Status values: `proposed`, `accepted`, `deprecated`, `superseded`
- Numbering: Increment `NNN` sequentially. Start from `001`.

Note: This repository uses `docs/internal/decisions/` for ADR storage (this folder).
